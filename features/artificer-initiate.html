Вимоги

Нет

Вы узнали кое-что из изобретательности ремесленника:

  • Вы изучаете один заговор и одно заклинание 1-го уровня по вашему выбору из списка заклинаний изобретателя [artificer]. Базовой характеристикой для этих заклинаний является Интеллект.
  • Вы можете наложить заклинание 1-го уровня используя эту черту не тратя ячейки заклинания. Вы должны закончить продолжительный отдых, прежде чем сможете накладывать заклинание полученные с помощью этой черты. Вы также можете накладывать эти заклинание, используя любые имеющиеся у вас ячейки заклинаний.
  • Вы получаете мастерство с одним типом инструментов ремесленника по вашему выбору, и вы можете использовать этот тип инструмента в качестве фокуса заклинания для любого заклинания, которое вы накладываете и которое использует Интеллект в качестве базовой характеристики заклинаний.
