Друг животных

Critter Friend

Джерело: Unearthed Arcana: Рассовые черты (UAFFR)

Вимоги

Эльф

Ваша дружба с животными усилена магией. Вы получаете следующие преимущества:

  • Вы получаете навык Уход за животными. Если он уже выбран вами, то ваш бонус мастерства удваивается для проверки этого навыка.
  • Вы изучаете заклинание разговор с животными [Speak with animals], которое можете использовать без затрат ячеек заклинаний. Вы также изучаете заклинание дружба с животными [Animal friendship], которое вы можете использовать один раз без затрат ячейки заклинаний. Вы можете вновь использовать заклинание бесплатно после продолжительного отдыха. Интеллект ваша базовая характеристика для этого заклинания.
